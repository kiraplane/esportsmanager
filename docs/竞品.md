## 1. 官方与一手来源

### Steam Store

- URL: https://store.steampowered.com/app/2749950/Esports_Manager_2026/
- 价值：发售日期、开发/发行信息、标签、系统需求、demo、achievements、review 状态、官方截图。
- 使用方式：作为 release date、demo、download、review、Steam Deck / PC FAQ 的主来源。

### Official Site

- URL: https://esportsmanager.gg/
- 价值：官方定位、核心玩法、社区链接、路线图、EMDB 入口、trailer。
- 使用方式：首页 facts、platforms、Discord/community、EMDB、roadmap FAQ。

### EMDB

- URL: https://emdb.gg/
- 价值：官方推荐的社区 roster/database 工具。
- 注意：需要 Steam 登录，并要求账号拥有游戏或 demo；不能复制或再分发用户导出的数据库。
- 使用方式：建立 `/emdb` 安全使用说明，而不是镜像数据库。

### Steam Community

- URL: https://steamcommunity.com/app/2749950
- 价值：官方公告、launch post、自定义数据库和 CustomAssets 路径说明、玩家早期问题。
- 使用方式：作为 EMDB / mod safety / patch note 的人工核查来源。

## 4. 竞品格局

本轮搜索没有发现成熟、稳定、可引用的大型 Esports Manager 2026 wiki/database 站点。官方 EMDB 是当前最重要的数据入口，Steam Community 与 YouTube 是早期攻略来源。

因此本站差异化策略：

1. 用首页承接 `wiki` 与 head terms。
2. 用 `/guides` 承接 guide/guides/tips。
3. 用 `/best-tactics` 承接 tactics/best tactics，但避免伪造阵容公式。
4. 用 `/emdb` 承接 database/roster 需求，引导到官方 EMDB，而不是复制数据。
5. 用 `/mods-and-cheats` 承接 cheats/mods/codes，明确安全边界。
6. 用 `/platforms` 解释 Steam / demo / mobile roadmap / console status。
